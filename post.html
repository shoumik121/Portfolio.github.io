Movie Ticketing Website

This was a project I started back in november 2019 when I first learned python. I was the sole creator/developer for the first year of the project before I was able to hire help.

The system stored fuel sample data for the few movies what were popular at that time and provided an interface for ticket sellers and admins to view, add and send data to customers.

I used Django to build that back end of this project & i went as i learned.

Customers had the ability to view available ticktes to their favourite shows and order them accordingly.

I worked on this system for 1 month before I started on a machine learning project.

Technologies:

  • - Django + Django
  • - Javascript
  • - Html
  • - Css
  • - Bootstrap
  • - Google Maps API
